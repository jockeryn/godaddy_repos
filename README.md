## Getting Started

This project is built in ReactJs using Vite framework and stylized with Boostrap.

### Prerequisites

Install NPM.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone git@github.com:jockeryn/godaddy_repos.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. For development
   ```sh
   npm run dev
   ```
5. Production build
   ```sh
   npm run build
   ```
6. Run tests
   ```sh
   npm run test
   ```
